# キヤクアシスト

マンション管理規約改正AIアシスタント — 改正区分所有法（2026年4月施行）に対応した規約改正をAIがステップバイステップで支援するウェブアプリケーション。

## 概要

2026年4月の改正区分所有法施行に伴い、全国約12万の管理組合が規約改正を迫られています。専門家に委託すると46〜58万円かかる一方、法律の素人である理事会メンバーが自力で進めるには知識・時間の壁が大きいのが現状です。

キヤクアシストは、AI（Claude API）を活用して「専門家と自力の間」を埋めるセルフサービス型の規約改正支援ツールです。

## 機能（6ステップ）

| # | 機能 | パス | 説明 |
|---|------|------|------|
| F0 | オンボーディング | `/onboarding` | マンション属性のヒアリング |
| F1 | 法改正かんたんガイド | `/guide` | FAQ形式で法改正のポイントを解説 |
| F2 | 規約アップロード | `/upload` | PDF/Word → AI解析 → 構造化 |
| F3 | ギャップ分析 | `/analysis` | 標準管理規約（令和7年改正）との差分分析 |
| F4 | 改正案レビュー | `/review` | カード型UIで1条文ずつ採否判断 |
| F6 | エクスポート | `/export` | PDF/Word 4形式でダウンロード |

## 技術スタック

- **フレームワーク**: Next.js 15 (App Router, TypeScript)
- **UI**: Tailwind CSS v4 + shadcn/ui
- **ホスティング**: Google Cloud Run（予定）
- **データベース**: Firestore（予定）
- **AI**: Claude API - Sonnet 4.5（予定）

## 開発

```bash
npm install
npm run dev     # http://localhost:3000
npm run build   # プロダクションビルド
```
